💮 Проект MAZE. 💮

Maze - он же Лабиринт, это программа для поиска кратчайшего пути из точки "А" в точку "Б". Представлена в виде игрового поля в виде лабиринта на котором пользователь указывает две точки ВХОД/ВЫХОД, а программа в ответ рисует кратчайший путь. Лабиринт можно сгенерировать, либо загрузить уже из готового файла.
Сгенерированный лабиринт в свою очередь отвечает всем правилам идеального лабиринта. А это значит что гарантированно не будет безвыходных участков, а так же циклично замкнутых пространств.

Установка и запуск:

  • Для простой установки предусмотрен "мэйкфайл" с целью install
  • После чего пользователь может открыть готовое приложение которое будет лежать в папке build
  • ВНИМАНИЕ: Для запуска тестов потребуется библиотека модульного тестирования gtest

Интерфейс программы:

  1. Кнопки открытия и сворачивания окна
  2. Интерфейс нативный и предельно понятный, с верху окна привычные нам кнопки открытия и закрытия.
  3. В центре у нас поле 500х500 пикселей.
  4. С права от игрового поля, центр управления, на котором раскинулись кнопки Load the Maze, Save the Maze, и Generate. Так же два поля Rows и Cols с помощью которых пользователь может указать размер будущего Лабиринта, затем если нажать на кнопку Generate на поле появится сгенерированный лабиринт, либо пользователь может загрузить заранее приготовленный лабиринт.
  5. Так же пользователь может сохранить в файл любой понравившейся сгенерированный лабиринт с помощью клавиши Save the Maze.

Makefile:

  • all
    • Произведет установку программы

  • install
    • Так же выполнит установку программы (не обязательно ее прописывать можно просто напечатать make)

  • uninstall
    • Удалит программу

  • test
    • Протестирует классы и функции написанные для работы модели.

  • dvi
    • Откроет данную документацию

  • dist
    • Выполнит создание tar-файла содержащего полный установочный пакет программы 3D-Viewer

  • clean
    • Удалит все ненужные и временые файлы

  • style
    • Проверит написанную библиотеку на google стиль
